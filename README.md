# AnyBody Project Page

Static project page for **AnyBody: Free-Form Whole-Body Humanoid Control from Arbitrary Keypoint Guidance**.

## Structure

- `index.html` contains the page content and local asset references.
- `static/teaser-video/` contains the teaser video.
- `static/method/` contains the method figure.
- `static/arbitrary-keypoint-guidance-following/` contains keypoint-guidance clips.
- `static/interactive-viewer/` contains the interactive viewer screenshot and video.
- `static/speed-commanded-locomotion/` contains locomotion clips.
- `static/wrist-writing/` contains in-air writing clips.
- `static/obstacle-reach/` contains obstacle-reach clips.
- `static/css/`, `static/js/`, and `static/webfonts/` contain page styling and script dependencies.

## Preview

Run a local static server from the repository root:

```bash
python3 serve.py 8000
```

Then open `http://localhost:8000`.
